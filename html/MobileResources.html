Web Resources

  1. Android Sensor Overview
    Sensor documentation created and maintained by Google for Android developers. Excellent information on what sensors are available and how to get to them.

  2. Arduino Accelerometer Projects
    Catalogue of various projects demoing what can be done with accelerometers. Helpful for coming up with ideas for mobile apps which make use of the accelerometer.

  3. Apple Developer Sensor Documentation
    Here is a list of sensors for the iphone including links to learn more about how to access their functionality as an apple developer.

  4. Amazon Developer Documentation for Fire OS
    How to create an Android mobile streaming app for FireTV. Useful documentation for generating ideas about how to interface with smart devices using android.
